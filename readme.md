# 🎸 MIDI Master Pro
Controlador MIDI DIY para pedaleiras, capaz de gerenciar 12 comandos através de apenas 4 footswitches físicos.

## 🚀 Destaques
- **Comandos Triplos:** Cada botão responde a Clique Simples, Duplo e Longo.
- **Toggle Mode:** Alterna entre dois valores configuráveis (Ex: Drive ON/OFF).
- **Interface Web:** Configuração rápida via USB sem tocar em código.
- **Presets JSON:** Salve e carregue setups para diferentes equipamentos.

## 📁 Arquivos
- `controlador.ino`: Firmware para o Arduino.
- `configuracao.html`: Interface de gerenciamento (Tailwind CSS).
- `HARDWARE.md`: Esquemas de solda e alimentação.

## 🛠️ Instalação Rápida
1. Suba o código `.ino` para o Arduino (31250 baud).
2. Abra `configuracao.html` no Chrome.
3. Conecte o USB, configure os números de CC/PC e grave no pedal.

[Guia Técnico de Montagem (Soldagem e 9V)](./HARDWARE.md)
